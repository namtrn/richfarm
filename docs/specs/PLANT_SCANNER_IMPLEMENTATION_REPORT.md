## Overview
The AI Plant Scanner allows users to identify plants from photos (camera or gallery) and receive a high-confidence match along with alternative suggestions if the detection is not 100% certain.

- **Primary AI**: `convex/plantScan.ts` now uses Gemini Vision (`gemini-2.5-flash`) via `generateContent`.
